2017 03 11 Retrieve

[Celtic defender Erik Sviatchenko says that losing 7-0 at Barcelona was an eye-opening experience] It was tough. To be fair, it was an experience even during the game. Now I can reflect back on it even more, but it’s never nice to be a part of a loss that big, but you also learn. I think you saw into the next game that we learned something about it, but from a personal perspective it’s all about the small details. You make one mistake, one pass, that might not go for a goal otherwise, it immediately is punished when you play against Neymar, Lionel Messi and Luis Suarez and so on. But it was a good experience and it was something I thought about during the game too.

It wasn’t a turning point because it was so early in the season, but it was an eye opener in some ways - we need to do much better if we want to be as good as we want to be. The potential is something we can dwell on, because it’s something that’s shown really quickly. It’s only been six months since the summer, when we started with the new team. What we have learned, where we are now, how we are playing, it’s been really good
